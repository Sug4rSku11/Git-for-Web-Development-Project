  1. What is Semantic HTML? 
        It is tags that describe tags with meaning.
    2. What is HTML used for? 
        It is a language used to display data in a graphical form.
    3. What is an attribute and where do we put it? 
        An attribute provides additional information about HTML elements and are always specified in the start tag.
    4. What is the h1 tag used for? How many times should I use it on a page?
        h1 is used for the most important heading and should be used once.

        <img>, <a>

        <title>, <meta>, <link>, <style>
    7. What is an id? 
        A unique identifier which must be unique in the whole document.
    8. What elements can I add an id to? 
        HTML elements
    9. How many times can I use the same id on a page? 
        One time.
    10. What is a class? 
        A title that can apply to multiple elements.

    12. How many times can I use the same class on a page? 
        multiple times
    13. How do I get my link to open in a new tab?
        target="_blank"
    14. What is the alt attribute in the image tag used for? 
        It provides an alternate text for an image.
    15. How do I reference an id?

    16. What is the difference between a section and a div
        A section represents a generic standalone section of a document. It should always have a heading.
        A div is the generic container for flow content. It has no effect on the layout until styled in CSS.
    17. What is CSS used for? 
        It is used to add style and layout to the webpage.

        A selector that selects elements that are in a specific state such as hover.
